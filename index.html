Тест на теги HTML5

1. Какой тег должен использоваться здесь, когда он находится в заголовке документа?
<?>Тег HTML5</?>

Выберите правильный ответ из предложенных вариантов

<header>
<h1>
<title>
Правильный ответ: <title>
Объяснение: Этот тег определяет заголовок документа, который отображается в заголовке браузера или на вкладке. Это обязательный элемент в HTML-документе.
2. Какой тег используется для создания абзаца текста?
Это абзац текста

Выберите правильный ответ из предложенных вариантов

<p>
<text>
<para>
Правильный ответ: <p>
Объяснение: Этот тег используется для создания абзацев текста. Браузер автоматически добавляет отступы (margin) сверху и снизу каждого абзаца.
3. Что делает тег <img> в HTML?

Выберите правильное определение из предложенных вариантов

Вариант 1

Создает интерактивную карту изображений с кликабельными областями

Вариант 2

Определяет контейнер для нескольких версий изображения для разных устройств

Вариант 3

Вставляет изображение на страницу с указанием источника через атрибут src

Правильный ответ: Вставляет изображение на страницу с указанием источника через атрибут src
Объяснение: Тег <img> - это одиночный тег (не требует закрывающего тега), который используется для вставки изображений на веб-страницу. Основные атрибуты:
  • src - обязательный атрибут, указывает путь к изображению
  • alt - альтернативный текст, отображается если изображение не загрузилось
  • width/height - задают размеры изображения
Пример использования: 
<img src="logo.png" alt="Логотип компании" width="200" height="100">
4. Какой тег используется для создания ненумерованного списка?

Выберите правильный ответ из предложенных вариантов

<ol>
  1. Пункт 1
  2. Пункт 2
<ul>
  • Пункт 1
  • Пункт 2
<dl>
Пункт 1
Подпункт
Правильный ответ: <ul>
Объяснение: Этот тег (Unordered List) создает ненумерованный (маркированный) список. Элементы списка помещаются в теги <li> и по умолчанию отображаются с маркерами в виде точек. 
<ul>
    <li>Пункт 1</li>
    <li>Пункт 2</li>
    <li>Пункт 3</li>
</ul>
5. Какой тег используется для создания секции в документе? 
<? id="chapter1">
                    <h2>Глава 1</h2>
                    <p>Текст первой главы...</p>
                </?>

Выберите правильный ответ из предложенных вариантов

<div>
<section>
<article>
Правильный ответ: <section>
Объяснение: Этот тег используется для создания тематической секции в документе. Это семантический тег HTML5, который помогает структурировать содержание страницы.
6. Какой тег используется для обозначения навигации?
<? class="menu">
                    <ul>
                    <li><a href="#">Главная</a></li>
                    <li><a href="#">О нас</a></li>
                    <li><a href="#">Контакты</a></li>
                    </ul>
                </?>

Выберите правильный ответ из предложенных вариантов

<navigation>
<menu>
<nav>
Правильный ответ: <nav>
Объяснение: Этот тег предназначен для обозначения основного блока навигационных ссылок. Это семантический тег HTML5, который помогает поисковым системам и скрин-ридерам понять структуру сайта.
7. Какой тег используется для создания формы?

Выберите правильный ответ из предложенных вариантов

<form>
<input>
<fieldset>
Правильный ответ: <form>
Объяснение: Этот тег создает форму для сбора пользовательских данных. Все элементы ввода (input, textarea, select и др.) должны находиться внутри этого тега, чтобы их данные могли быть отправлены на сервер.
8. Какой тег используется для добавления ссылки?

Выберите правильный ответ из предложенных вариантов

<link>
<a>
<href>
Правильный ответ: <a>
Объяснение: Этот тег (anchor - якорь) создает гиперссылку. Атрибут href определяет URL-адрес, на который ведет ссылка. Это один из самых важных тегов в HTML.
9. Какой тег используется для вставки видео?

Выберите правильный ответ из предложенных вариантов

<media>
<video>
<movie>
Правильный ответ: <video>
Объяснение: Этот тег позволяет встраивать видеофайлы на веб-страницу. Поддерживает несколько форматов видео через теги <source>. Имеет встроенные элементы управления воспроизведением.
